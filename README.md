# Vocabulary with AI 🚀

Gemini API를 활용하여 기본 단어 100개를 추천해주는 10개 국어 단어장입니다.
영어, 중국어, 힌디어, 스페인어, 프랑스어, 아랍어, 벵골어, 포르투갈어, 러시아어, 일본어 단어장을 만들 수 있습니다.

## 👥 데이터 저장 방식

- **기능 공유**: 배포된 URL로 접속하는 모든 사용자는 배포자가 설정한 API 키를 통해 AI 기능을 사용할 수 있습니다.
- **개별 데이터**: 모든 단어 데이터는 사용자의 브라우저(`localStorage`)에 저장됩니다. **사용자 간에 단어 리스트는 공유되지 않으며**, 각 접속자는 자신만의 독립된 단어장을 관리하게 됩니다.

## ✨ 주요 기능

- **AI 추천 단어 생성**: 기본 100단어 즉시 로드
- **가리기 모드**: 가리기 기능을 통해 암기가 되었는지 바로 테스트할 수 있습니다.
- **실시간 검색**: 구글 검색에 기반해 새로운 단어를 추가할 시 예문도 함께 생성해줍니다.
- **즐겨찾기 및 CSV 지원** : 엑셀로 단어를 일괄 등록 할 수 있습니다.
